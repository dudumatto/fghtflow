# SKILLS.md (Agent Kit)

## Objetivo

Skills curtas e reutilizáveis para execução de qualidade, independentes de stack específica.

## Skills disponíveis

- Design system → `skills/design-system.md`
- UI accessibility (a11y) → `skills/ui-accessibility.md`
- Responsividade → `skills/responsiveness.md`
- Motion & feedback → `skills/motion-and-feedback.md`
- Frontend best practices → `skills/frontend-best-practices.md`
- Backend best practices → `skills/backend-best-practices.md`
- Segurança → `skills/security.md`
- Testes → `skills/testing.md`
- Documentação → `skills/documentation.md`
- Performance → `skills/performance.md`

## Como usar

1. Selecione os **agentes** (ver `AGENTS.md`).
2. Aplique as **skills** correspondentes enquanto implementa.
3. Use `rules/COLLABORATION.md` para organizar o fluxo e `rules/OUTPUT_FORMAT.md` para padronizar a entrega.
