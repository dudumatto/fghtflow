# Skill: Backend Best Practices

## Objetivo
APIs consistentes, seguras e compatíveis.

## Práticas (curtas)
- Validar entradas (tipo, tamanho, enum, null).
- Erros consistentes (status code + mensagem).
- Não quebrar contratos sem migração.
- Paginação/limites para listagens.
- Separação clara de camadas (controller/service/repo).
