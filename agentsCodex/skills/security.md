# Skill: Segurança

## Objetivo
Reduzir riscos comuns (auth, autorização, injeção, vazamento de dados).

## Práticas (curtas)
- Autorização no servidor (ownership/escopo).
- Validar e sanitizar inputs (principalmente strings).
- Evitar logs com dados sensíveis.
- Proteções contra XSS/CSRF/IDOR conforme o tipo de app.
- Uploads: validar tipo/tamanho e armazenar com segurança.
